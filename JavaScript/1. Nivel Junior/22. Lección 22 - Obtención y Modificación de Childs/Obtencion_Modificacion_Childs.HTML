Lección 22 - Obtencion y Modificacion de Childs

Definición: Metodos que permiten acceder a elementos hijos de elementos padre
Metodos:

  • elementoPadre.firstChild Toma el primer hijo del elemento padre, toma en cuenta texto que tambien alla
  • elementoPadre.lastChild Toma el ultimo hijo del elemento padre, incluye texto si hay
  • elementoPadre.firstElementChild Toma el primer elemento hijo del elemento padre
  • elementoPadre.lastEelementChild Toma e ultimo elemento hijo del elemento padre
  • elementoPadre.childNodes Toma todos los hijos del elemento padre, incluyendo texto
  • elementoPadre.children Toma todos los elementos hijos del elemento padre.
