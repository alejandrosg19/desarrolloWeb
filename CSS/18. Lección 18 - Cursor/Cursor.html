Lección 18 - Cursor

Definición: Es una propiedad que permite cambiar el estilo del cursor cuando este pasa por encima del elemento que seleccionamos para cambiar el estilo del cursor

Ejemplo: pasa por encima y cambia el estilo el curso y si mantienes presionado tambien cambiara el estilo de cursor
