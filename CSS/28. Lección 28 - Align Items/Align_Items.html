Lección 28 - Align Items - Align Content

Defnición: Estas dos propiedades permiten alineas los elementos contenidos dentro de un flex-container o los flex-items sobre el eje cross-axis
align-item: Esta propiedad se usa solo cuando hay una sola linea de elementos o de flex-items
align-content: Esta propiedad se usa cuando hay mas de una linea de elementos o de flex-items
Cabe destacar que ambos poseen los mismos tipos de propiedades.

Tipos de align-content y align-item

align-item: stretch Esta propiedad se encarga de alinear en la parte superior los elementos en el eje cross-axis o eje y, cabe destacar que si los elementos no posee un largo, con esta propiedad los elementos van a tomar el largo del contenedor.
