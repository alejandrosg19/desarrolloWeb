Lección 52 - Grid Area

Definición: Grid Area se le dice a un conjunto de grid-celdas seleccionadas consecutivamente de la grilla, deben ser mas de 2.
grid-template-area: Esta propiedad se usa para definir las areas de las grilla. se usa con nombres en de las celdas, por ejemplo.
grid-template-area:
"header header header"
aside main main"
aside main main"
footer footer footer".
Esto da como resultado:

Ahora bien para poder asignar los divs que son grid-items a cada de una de las areas, se hace por medio de grid-area: "nombre del area", ejemplo grid-area: header;
