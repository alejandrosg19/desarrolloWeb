Lección 22 - Introducción Flexbox

Definición: Cajas flexibles.

Conceptos:

flex-container: Es el que contendra los diferentes elementos

flex-item: Son los items elementos dentro del flex container
Cuando aplicamos display: flex, el elemento se convierte en un elemento de bloque, como si estuviesemos dandole un display: block; pero el verdadero cambio lo veremos es dentro del elemento o contenedor al que le aplicado la propiedad de flex. Osea a los items

Ejes: Flexbox posee dos ejes, como si tuveramos el eje x y el eje y en un plano cartesiano. Estos ejes posee una dirección

  • Main axis -> Eje x eje horizontal y posee main-start y main-end

  • Cross axis -> Eje y eje vertical y posee cross-start y cross-end
