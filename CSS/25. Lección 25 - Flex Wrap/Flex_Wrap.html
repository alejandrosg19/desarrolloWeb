Lección 25 - Flex Wrap

Definición: Es una propiedad que respeta el ancho de los elementos dentro del flex-container o los flex-items, es deir, en vez de que los elementos se angosten a medida que la ventana se hace mas angosta, los elementos se van poniendo uno encima de otro respetando el ancho de estos.

Tipos de flex-wrap

Sin wrap: Propiedad por defecto en donde los elementos se van angostando según el ancho de la ventana
