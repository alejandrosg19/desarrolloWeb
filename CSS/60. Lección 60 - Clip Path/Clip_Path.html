Lección 60 - Clip Path

Definición: Es una propiedades que nos permite modelar la figura de un elemento, podemos convertir un elemento en un triangulo, una bota un barco, etc. Cabe destacar que esta propiedad usa coordenadas y existen algunor generadores de las coordenadas según la figura que necesite.
