Lección 23 - Display Flex

Definición cuando aplicamos flex a un div o elemento, los elementos dentro de este se posicionan uno al lado del otro, con un tamaño en base a su contenido. Pero ese tamaño es solo afectado en el ancho, el largo de los elementos va a hacer el mismo, independientemente de si uno tiene mas contenido que el otro. Tambien cabe destacar que el flex-container solo influye en los elemenos hijos directamente, no en los elementos que estan dentro de los elementos que son sus hijos. Ejemplo:
