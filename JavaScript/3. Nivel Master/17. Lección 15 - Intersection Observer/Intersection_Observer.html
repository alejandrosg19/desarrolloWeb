Lección 15 - Intersection Observer

Definición: La API Observador de Intersección provee una vía asíncrona para observar cambios en la intersección de un elemento con el viewport, es decir que, es un objeto que vigilia si un elemento se en cuentra en el viewport
Objeto: El objeto se crea de la siguiente manera: const observer = new IntersectionObserver(callback, obtions); cabe destacar que el callback siempre recibe un arreglo, el segundo parametro corresponde a las opciones
Opciones:

  • root: Elemento de referencia
  • rootMargin: margin que se aplicara en el viewpart para informar cuando se esta viendo un elemento
Metodos:
  • obj_observer.observe(objeto): Recibe un elemento el cual queremos que sea observado.
Tener en cuenta: Cada vez que uno de los elementos que se escogieron paa observar para por el viewport, se activa el callbacl o funcion con que se creo el observer
