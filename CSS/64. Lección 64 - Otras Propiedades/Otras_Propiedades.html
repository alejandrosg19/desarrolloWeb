Lección 64 - Otras Propiedades

  • direction: ltr | rtl | initial | inherit Esta propiedad permite cambiar la dirección de los elementos.
    • ltr left to rigth
    • rtl rigth to left
  • letter-spacing: number px Esta propiedad separa las letras la cantidad de pixeles que nosotros queramos.
  • user-select: none Esta propiedad tiene como objetivo no permitirle al usuario seleccionar el texto. Esto significa que el usuario no podracopiar la información.
  • scroll-behavior: smooth Esta propiedad funciona cuando utilizamos una etiqueta a para hacer scroll en la misma pagina y llevar al usuario a una parte especifica de la pagina, podemos hacer que suba o que baje, la cuestion es que con la propiedad dicho movimiento no se hace de impacto si no que hay un transición.
