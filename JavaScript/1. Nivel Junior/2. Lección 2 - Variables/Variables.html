Lección 2 - Variables

Definición: Es una espacio que se guarda en memoria
Para definiri una vaiable simplement colocamos el nombre de la Variable seguido de = y seguido del contenido de la variable, por ejemplo:
nombre = "Santiago";
nombre = "Alejandro";

Existe tres maneras de declarar una variable:
  • var: Es una variable global que puede ser usada desde cualquier condicional o metodo
  • let: Es una variable que solo puede ser usada dentro del contenedor donde fue declarada, es decir, si dentro de un if yo declare mi variable let, solo puedo hacer uso de dicha variable dentro de ese if, dentro de ese bloque de codigo, por fuera no.
  • constat: Hace referencia a una variable constate, significa que una vez declarada, la variable siempre va a tomar ese valor, no se lo podremos cambiar. Si itentamos cambiar una variable const, nos va a arrojar error. Cabe descatar que con const es obligatorio inicializarla en el mismo instante en que la declaramos.

Proceso de declaración, inicialización y modificación

  • Declaración: let numero;
  • Inicialización: numero = 19;
  • modificación: let numero = 19;
    numero = 20;

Inicialización multiple de variables

  • let variable1, variable2, variable3
  • let variable1 = 1, variable2 = 2, variable3 = 3

Tipos de datos:
  • string: Cadena de texto Ejemplo: string = "Cadena de texto"
  • number: Numero 2 Ejemplo number = 19
  • booleanos: True o False bool = true
  • Casos especiales de datos: Estos tipos de variables nos habla de que la variable no esta definida o hay un error
    • undefined: Es una variable que ha sido declarada pero no definidia o inicializada, es decir, que no tiene valor.
    • null: Variable que no tiene nada, a diferencia del undefined es que la variable es declarada pero tambien es inicializada con un valor nulo
    • nan: Significa: No es un numero, esto resulta cuando intentamos realiar una operación entre dos variables es la al menos una no es un numero.

Scope: Cuando hablamos de scope hablamos del alcance que tiene el tipo de variable en el bloque de codigo, por eso hacemos referencia a que el tipo de variable let, tiene un alcance local, dentro del bloque de codigo en que se declaro.
alert(): alert es una función que nos arroja un pop up en pantalla con cualquier valor que nosotros queremos mostrar.
prompt(): Tiene la misma funcionalidad que alert solo que ademas de mostrar tambien tiene un recuadro en el que el usuario puede ingresar valores, basicamente podemos campturar valores. Por ejempo:
var nombre = prompt("Dame tu nombre"); con la siguiente instrucción estamos mostrando un mensaje "Dame tu nombre" y ademas almacenando el valor que el usuario envie en la variable nombre.
