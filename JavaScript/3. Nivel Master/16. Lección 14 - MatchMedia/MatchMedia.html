Lección 14 - MatchMedia

Definición:Es una api de JavaScript que permite trabajar con responsive design, esta api es buena usarla cuando hay cosas que no podemos hacer en CSS. Cabe destacar que se usa generalmente para meter funcionalidades no para cambiar estilos
Creación de objeto: const mql = matchMedia("(max-widt: 500px)")
Evento mql.addEventListener("change" ()=>{}): Se activa cuando mql.matches cambia, es decir, cuando se cumple o no se cumple la el tamaño especificado dentro dem matchMedia("") cuando creamos el objeto.
