Lección 21 - Creación de Elementos

Definición: Permite crear elementos html dentro de elementos ya definidos
Metodos:

  • document.createElements("elemento HTML") Crea un elemento HTML definido dentro de los parentesis. Es bueno usarlo para cuando es un solo elemento el que se quiere agregar.
  • document.createTextNode("texto") Crea un texto dentro del elemento
  • document.createDocumentFragment("elemento HTML") Es una manera optima de crear un fragmento HTML con varios elementos dentro sin necesidad de utilizar createElements() y hacer uso de muchos recursos
  • elemento.appendChild("Elemento HTML") Permite agregar a un elemento contenido HTML
