Lección 16 - Pseudo Clases

Definición: Las Pseudo-Clases escuchan eventos y cuando dicho evento se activa, se llevan a cabo los efectos aplicados a dicho evento

Tipos de Pseudo-Clases

:hover Es una propiedad que permite aplicarle un cambio de efecto a un elemento cuando el mouse pase por encima de este. Cabe destacar que con hover se agregan otras fucionalidades como la del transition, en la que podemos especificar que tan rapido queremos que se den esos cambios en el hover para cada uno de las propiedaes que queramos. Ejemplo de hover, pasa el cursor por encima

:link Es una Pseudo-Clase que le da propiedades a un link que aun no es visitado o clickeado por el usuario. Va a mantener unas propiedades definidias siempre y cuando el link al qe hace referencia el a no se haya visitado, si ha visitado al menos una vez, el elemento a pierde las propiedades aplicadas con :link Ejemplo:

Si no me cliqueas soy de un color, pero cuando me cliquees cambio de color

:visited Es una Pseudo-Clase similar a la anterior, la diferencia es que esta permite aplicarle propiedades al elemento a osea a un link que ya fue visitado. Ejemplo:

Soy un link con las dos propiedades de link y visited

:lang() Es una Pseudo-Clase que aplica propiedades a un texto de un lenguaje en especifico, es decir podemos decirle que aplique certas propiedades a un texto si este se encuentra en ingles, pero para eso, es necesesario darle en el HTML al texto, el lenguaje al que corresponde, por medio de la etiqueta lang="es" -> para español o lang="en" -> para ingles.
