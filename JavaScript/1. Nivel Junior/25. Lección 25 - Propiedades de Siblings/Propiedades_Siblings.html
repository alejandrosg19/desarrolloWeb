Lección 25 - Propiedades de Siblings (Hermanos)

Definición: Los elementos hermanos indepedientemente de que se han diferentes tipos de elementos, son hermanos por el hecho de estar en la misma linea.
Propieades:

  • nextSibling: Devuelve el nodo siguiente del elemento seleccionado, incluye textNode (texto)
  • previousSibling Devuelve el nodo anterior del elemento seleccionado, incluye textNode (texto)
  • nextElementSibling Devuelve el elemento siguiente del elemento seleccionado
  • previousElemenetSibling Devuelve el elemento anterior del elemento seleccionado
