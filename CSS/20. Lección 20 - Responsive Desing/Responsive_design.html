Lección 20 - Responsive Designs

Definición: Responsive design en la manera en que diseñamos una pagina web para que visualmente se vea de manera correcta o que visualmente se vea agradable a los usuarios independientemente de l dispositivo en el que se encucnetren visualizando la pagina.

Herramientas:

Etiqueta meta:

meta name="viewport" content="width=device-width, initial-scale=1.0" esta etiqueta permite darle a la pagina web propiedades que le hagan ser responsive

Etiqueta @media:

@media only screen and (max-width: 800px){ div{display: block;}} esta etiqueta se usa en CSS y sirve para cambiar las propiedades a los elementos para un tamaño de ventana o para un tipo de dispositivo, por ejemplo en el anterior @media, decimos que cuando la ventana llegue a menos de 800px todos los div de la pagina web a cambiar su propiedad display a block.
