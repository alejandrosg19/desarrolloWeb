Lección 19 - Colores

A continuación observaremos las distintas formas de aplicar colores a los elementos

Formas de aplicar colores:

name: Una de las formas de poder aplicar un color es dar el nombre del color directamente. Ejemplo

color: red; background-color: yellow;

rgb Significa Red, Green, Blue y lo que hace es mezclar estos tres tipos de colores para dar una gran variedad de colores y como se basa en estos tres colores, al momento de nosotros aplicar un color a alguna propiedad con RGB, se deben llenar tres campos que corresponden a la cantidad de cada uno de los tres colores que se necesita para llegar al color que queremos. 0 indica no aplicar nada de ese color y 255 Significa aplicar todo de ese color. rgb(0,4,10); Ejemplo

Color rojo -> color: rgb(255,0,0);
Color de fondo verde-> background-color: rgb(0,255,0);

rgba Es lo mismo que el rgb, la unica diferencia es que posee un cuarto parametro que es el "a" que signfica alpha y corresponde a la opacidad del color rgba(255,0,0,.5). Ejemplo:

Color rojo -> color: rgb(0,0,255,.3);
Color de fondo amarillo-> background-color: rgb(255,255,.6);

hexadecimal Es un sistema de numeración de 16 unidades, que funciona similar al rgb pero con letras y numeros, si por ejemplo queremos el color blanco podremos escribirlo de dos formas "#fff" en donde cada f significa 255 de los tres colores o "#ffffff" en donde cada 2 f corresponde a uno de los tres colores, es decir, las primeras dos ff son del rojo, las siguientes dos ff corresponden al ver y las ultimas al azul. Cabe destacar que tambien se puede color cuatro numeros "#ffff" para representar el rgba o con 8 numeros "#ffffffff". color: #f00; Ejemplo:

Con tres y seis valores

Color rojo -> color: #f00; o #ff0000
Color de fondo verde-> background-color: #0f0 o #00ff00;

Con 4 y ocho valores

Color rojo -> color: #f003;
Color de fondo amarillo-> background-color: #ffff0060;
