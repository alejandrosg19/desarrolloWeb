Lección 54 - Responsive Design

Definición: Es un concepto que se usa para hacer que una pagina web pueda ser vista adecuadamente en diferentes tamaños de dispositivos. Dentro del Responsive Design en CSS se hace uso de @media
@media Significa media query y se usa en consultas de medios para aplicar diferentes estilos para diferentes tipos de dispositivos. Basicamente es un conjunto de condicionales que nos permite determinar el cambio de diseño de una pagina y alicarle propiedades a dicho cambio de diseño. El media query nos permite validar:

  • Ancho y alto de la ventana grafica
  • ancho y alto del dispositivos
  • Orientación (tablet -> Modo horizontal - Telefono -> Modo Vertical)
  • Resolución
Tipos de media query
  • all Apto para todos los dispositivos
  • print Destinado a material impreso y visualización de documentos en una pantalla en el modo de vista previa de impresión
  • screen Destinado principalmente a las pantallas
  • speech Destinado principalmente a sitetizadores de voz
Operadores
  • AND
  • OR
Vamos a ver el funcionamiento de screen, ejemplo
@media screen and (min-width: 500px) and (max-width: 800px){
div{
background-color: red;
}
}
El anterior media screen valida si la ventana se enceuntra en un tamaño entre 500px y 800px, de ser asi aplicara las propiedades designadas, es decir cambiara el color de fondo de todos los divs a rojo.
Ejemplo, para visuaizar los cambios, por favor juegue con el tamaño de la ventana.
